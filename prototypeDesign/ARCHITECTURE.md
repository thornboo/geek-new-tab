# 代码架构说明

## 文件组织

### index.html
- **作用**: 应用的主入口文件
- **结构**: 
  - 头部时钟和搜索框
  - 左侧分类导航栏
  - 右侧网站卡片区域
  - 弹窗组件（添加内容、设置）

### styles.css
- **作用**: 所有样式定义
- **特点**:
  - CSS变量系统，支持主题切换
  - 响应式设计，适配移动端
  - 3D翻转动画效果
  - 极客风格的视觉设计

### config.js
- **作用**: 配置和默认数据
- **内容**:
  - 默认网站数据结构
  - 默认用户配置
  - 存储键名常量

### app.js
- **作用**: 核心应用逻辑
- **模块**:
  - 数据管理（加载/保存）
  - 界面渲染（分类/卡片）
  - 用户交互（搜索/快捷键）
  - 功能模块（分类管理/网站管理）

## 数据流向

```
localStorage ←→ 全局状态 ←→ 界面渲染
     ↑              ↑           ↑
   持久化        数据处理     用户交互
```

## 核心功能模块

### 1. 分类管理
- 添加分类：`commitNewCategory()`
- 编辑分类：`editCategory()`
- 删除分类：`deleteCategory()`
- 切换分类：`switchCategory()`

### 2. 网站管理
- 添加网站：`commitNewSite()`
- 编辑网站：`editSite()`
- 删除网站：`deleteSite()`
- 访问统计：`trackSiteVisit()`

### 3. 搜索功能
- 站内搜索：`showSiteSearchResults()`
- 实时过滤：基于名称、描述、标签
- 快速访问：Enter键直达

### 4. 界面渲染
- 分类导航：`renderTabs()`
- 网站网格：`renderGrid()`
- 空状态处理：自动显示提示

### 5. 用户设置
- 主题配置：颜色、背景
- 数据管理：备份、重置
- 搜索引擎：自定义引擎

## 技术特点

### 动画系统
- 使用 Anime.js 实现流畅动画
- 卡片3D翻转效果
- 平滑的状态转换

### 响应式设计
- 桌面端：侧边栏布局
- 移动端：顶部导航
- 自适应网格系统

### 数据持久化
- localStorage 本地存储
- 自动保存用户数据
- 配置与数据分离

### 用户体验
- 快捷键支持
- 实时搜索反馈
- 空状态友好提示
- 操作确认机制

这个架构确保了代码的可维护性和扩展性，为后续功能开发提供了良好的基础。
